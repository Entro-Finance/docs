# API Keys

Manage API keys for secure access to the Entro Finance platform.

## GET /api/auth/keys

List all API keys for the authenticated user.

<ParamField header="Authorization" type="string" required>
  Bearer authentication header of the form Bearer &lt;token&gt;, where &lt;token&gt; is your auth token.
</ParamField>

<ParamField query="page" type="string" default="1">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="string" default="25">
  Number of results per page
</ParamField>

<ResponseExample>
```json Response
{
  "apiKeys": [
    {
      "id": "key_7b8a9c0d1e2f3456",
      "name": "Production API Key",
      "keyHash": "sk_live_...f3456",
      "permissions": ["cards:read", "cards:write", "transactions:read"],
      "lastUsed": "2024-01-15T10:30:00Z",
      "createdAt": "2024-01-01T00:00:00Z",
      "expiresAt": "2025-01-01T00:00:00Z",
      "status": "active"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 25,
    "total": 3,
    "totalPages": 1
  }
}
```
</ResponseExample>

<RequestExample>
```bash cURL
curl --request GET \
  --url https://api.entrofinance.com/api/auth/keys \
  --header 'Authorization: Bearer sk_live_...'
```

```javascript JavaScript
const response = await fetch('https://api.entrofinance.com/api/auth/keys', {
  headers: {
    'Authorization': 'Bearer sk_live_...',
    'Content-Type': 'application/json'
  }
});
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.entrofinance.com/api/auth/keys',
    headers={
        'Authorization': 'Bearer sk_live_...',
        'Content-Type': 'application/json'
    }
)
data = response.json()
```
</RequestExample>

## POST /api/auth/keys

Generate a new API key with specified permissions.

<ParamField header="Authorization" type="string" required>
  Bearer authentication header of the form Bearer &lt;token&gt;, where &lt;token&gt; is your auth token.
</ParamField>

<ParamField body="name" type="string" required>
  Name for the API key
</ParamField>

<ParamField body="permissions" type="array" required>
  Array of permission strings for the key
</ParamField>

<ParamField body="expiresIn" type="string" default="1y">
  Expiration time (e.g., "30d", "1y", "never")
</ParamField>

<ResponseExample>
```json Response
{
  "apiKey": {
    "id": "key_8c9b0a1d2e3f4567",
    "name": "Development Key",
    "key": "sk_test_8c9b0a1d2e3f4567890abcdef1234567",
    "permissions": ["cards:read", "transactions:read"],
    "createdAt": "2024-01-15T10:30:00Z",
    "expiresAt": "2025-01-15T10:30:00Z",
    "status": "active"
  }
}
```
</ResponseExample>

<RequestExample>
```bash cURL
curl --request POST \
  --url https://api.entrofinance.com/api/auth/keys \
  --header 'Authorization: Bearer sk_live_...' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "Development Key",
    "permissions": ["cards:read", "transactions:read"],
    "expiresIn": "1y"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.entrofinance.com/api/auth/keys', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer sk_live_...',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Development Key',
    permissions: ['cards:read', 'transactions:read'],
    expiresIn: '1y'
  })
});
```

```python Python
import requests

response = requests.post(
    'https://api.entrofinance.com/api/auth/keys',
    headers={
        'Authorization': 'Bearer sk_live_...',
        'Content-Type': 'application/json'
    },
    json={
        'name': 'Development Key',
        'permissions': ['cards:read', 'transactions:read'],
        'expiresIn': '1y'
    }
)
```
</RequestExample>

## DELETE /api/auth/keys/{keyId}

Revoke an API key permanently.

<ParamField header="Authorization" type="string" required>
  Bearer authentication header of the form Bearer &lt;token&gt;, where &lt;token&gt; is your auth token.
</ParamField>

<ParamField path="keyId" type="string" required>
  The ID of the API key to revoke
</ParamField>

<ResponseExample>
```json Response
{
  "success": true,
  "message": "API key revoked successfully",
  "revokedAt": "2024-01-15T10:30:00Z"
}
```
</ResponseExample>

<RequestExample>
```bash cURL
curl --request DELETE \
  --url https://api.entrofinance.com/api/auth/keys/key_7b8a9c0d1e2f3456 \
  --header 'Authorization: Bearer sk_live_...'
```

```javascript JavaScript
const response = await fetch('https://api.entrofinance.com/api/auth/keys/key_7b8a9c0d1e2f3456', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer sk_live_...'
  }
});
```

```python Python
import requests

response = requests.delete(
    'https://api.entrofinance.com/api/auth/keys/key_7b8a9c0d1e2f3456',
    headers={
        'Authorization': 'Bearer sk_live_...'
    }
)
```
</RequestExample>

## Available Permissions

| Permission | Description |
|------------|-------------|
| `cards:read` | Read card information and details |
| `cards:write` | Create and modify cards |
| `transactions:read` | Read transaction history |
| `transactions:write` | Process transactions |
| `wallets:read` | Read wallet information |
| `wallets:write` | Modify wallet settings |
| `webhooks:read` | Read webhook configurations |
| `webhooks:write` | Create and modify webhooks |
| `analytics:read` | Access analytics and reporting data |
