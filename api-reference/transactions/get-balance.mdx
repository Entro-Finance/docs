# Get Balance

Retrieve current balance information for the authenticated account and associated cards.

## GET /api/balance

Get comprehensive balance information including all cards and pending transactions.

<ParamField header="Authorization" type="string" required>
  Bearer authentication header of the form Bearer &lt;token&gt;, where &lt;token&gt; is your auth token.
</ParamField>

<ParamField query="cardId" type="string" optional>
  Get balance for specific card only
</ParamField>

<ParamField query="currency" type="string" optional>
  Filter by currency: `USD`, `EUR`, `GBP`
</ParamField>

## Balance Types

| Type | Description |
|------|-------------|
| `available` | Immediately spendable balance |
| `pending` | Funds being processed or settled |
| `reserved` | Funds reserved for pending transactions |
| `total` | Sum of available + pending + reserved |
