# List Transactions

Retrieve a paginated list of all transactions associated with the authenticated account.

## GET /api/transactions

List all transactions with optional filtering and pagination.

<ParamField header="Authorization" type="string" required>
  Bearer authentication header of the form Bearer &lt;token&gt;, where &lt;token&gt; is your auth token.
</ParamField>

<ParamField query="page" type="string" default="1">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="string" default="25">
  Number of results per page (max 100)
</ParamField>

<ParamField query="cardId" type="string" optional>
  Filter by specific card ID
</ParamField>

<ParamField query="type" type="string" optional>
  Filter by transaction type: `payment`, `funding`, `withdrawal`, `refund`
</ParamField>

<ParamField query="status" type="string" optional>
  Filter by status: `pending`, `completed`, `failed`, `cancelled`
</ParamField>

<ParamField query="dateFrom" type="string" optional>
  Start date filter (ISO 8601 format)
</ParamField>

<ParamField query="dateTo" type="string" optional>
  End date filter (ISO 8601 format)
</ParamField>

## Transaction Types

| Type | Description |
|------|-------------|
| `payment` | Card payment at merchant or online |
| `funding` | Adding funds to card from wallet |
| `withdrawal` | Withdrawing funds from card to wallet |
| `refund` | Merchant refund or transaction reversal |
| `fee` | Platform or network fees |
| `reward` | ENTRO token rewards or cashback |

## Transaction Status Values

| Status | Description |
|--------|-------------|
| `pending` | Transaction is being processed |
| `completed` | Transaction completed successfully |
| `failed` | Transaction failed due to error |
| `cancelled` | Transaction was cancelled |
| `declined` | Payment declined by network or issuer |
