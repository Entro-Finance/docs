# API Quickstart

Get up and running with the Entro Finance API in just a few minutes.

## Step 1: Get Your API Key

1. Sign in to your [Entro Finance Dashboard](https://dashboard.entrofinance.com)
2. Navigate to **API Keys** in the settings
3. Click **Generate New Key**
4. Copy your API key securely

## Step 2: Make Your First Request

<CodeGroup>

```javascript Node.js
const response = await fetch('https://api.entrofinance.com/api/v1/balances', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const balances = await response.json();
console.log(balances);
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get('https://api.entrofinance.com/api/v1/balances', headers=headers)
balances = response.json()
print(balances)
```

```curl cURL
curl -X GET "https://api.entrofinance.com/api/v1/balances" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

</CodeGroup>
