# Connect Wallet

Connect a Solana wallet to enable crypto card funding and management.

## POST /api/wallets/connect

Connect a Solana wallet address to the authenticated account.

<ParamField header="Authorization" type="string" required>
  Bearer authentication header of the form Bearer &lt;token&gt;, where &lt;token&gt; is your auth token.
</ParamField>

<ParamField body="walletAddress" type="string" required>
  Base58 Solana wallet address
</ParamField>

<ParamField body="signature" type="string" required>
  Signed message proving wallet ownership
</ParamField>

<ParamField body="message" type="string" required>
  Original message that was signed
</ParamField>

<ParamField body="provider" type="string" optional>
  Wallet provider: `phantom`, `solflare`, `backpack`, `ledger`
</ParamField>

## Supported Wallets

| Wallet | Provider | Type |
|--------|----------|------|
| Phantom | `phantom` | Browser Extension/Mobile |
| Solflare | `solflare` | Browser Extension/Mobile |
| Backpack | `backpack` | Browser Extension |
| Ledger | `ledger` | Hardware Wallet |

## Message Signing

The message to sign should follow this format:
```
Connect wallet to Entro Finance - {ISO_TIMESTAMP}
```

Example: `Connect wallet to Entro Finance - 2024-01-15T10:30:00Z`
