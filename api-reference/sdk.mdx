# SDKs & Libraries

Official and community-maintained libraries to integrate with Entro Finance.

## Official SDKs

### JavaScript/TypeScript

```bash
npm install @entrofinance/sdk
```

```javascript
import { EntroClient } from '@entrofinance/sdk';

const client = new EntroClient({
  apiKey: 'your-api-key',
  environment: 'production' // or 'sandbox'
});

// Get account balances
const balances = await client.balances.list();
```

### Python

```bash
pip install entrofinance
```

```python
from entrofinance import EntroClient

client = EntroClient(api_key='your-api-key')
balances = client.balances.list()
```

## Community Libraries

- **Go**: `github.com/entrofinance/go-sdk`
- **PHP**: `composer require entrofinance/php-sdk`  
- **Ruby**: `gem install entrofinance`

## Features

All SDKs include:
- Type-safe API client
- Automatic retry logic
- Rate limit handling
- Error handling
- WebSocket support
