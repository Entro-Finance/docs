# Create Card

Create a new virtual or physical card linked to a Solana wallet for crypto spending.

## POST /api/cards

Create a new card with specified configuration and funding options.

<ParamField header="Authorization" type="string" required>
  Bearer authentication header of the form Bearer &lt;token&gt;, where &lt;token&gt; is your auth token.
</ParamField>

<ParamField body="type" type="string" required>
  Card type: `virtual` or `physical`
</ParamField>

<ParamField body="currency" type="string" required>
  Card currency: `USD`, `EUR`, `GBP`
</ParamField>

<ParamField body="name" type="string" required>
  Display name for the card
</ParamField>

<ParamField body="walletAddress" type="string" required>
  Solana wallet address for funding
</ParamField>

<ParamField body="limits" type="object" required>
  Card spending limits configuration
  <Expandable>
    <ParamField body="daily" type="number" required>
      Daily spending limit in card currency
    </ParamField>
    <ParamField body="monthly" type="number" required>
      Monthly spending limit in card currency
    </ParamField>
    <ParamField body="perTransaction" type="number" required>
      Maximum per-transaction limit
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="fundingSource" type="object" optional>
  Initial funding configuration
  <Expandable>
    <ParamField body="token" type="string" required>
      Token to fund with: `SOL`, `USDC`, `USDT`, `BTC`, `ETH`
    </ParamField>
    <ParamField body="amount" type="number" required>
      Amount to fund in token units
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="shippingAddress" type="object" optional>
  Required for physical cards only
  <Expandable>
    <ParamField body="name" type="string" required>
      Recipient full name
    </ParamField>
    <ParamField body="street" type="string" required>
      Street address
    </ParamField>
    <ParamField body="city" type="string" required>
      City
    </ParamField>
    <ParamField body="state" type="string" required>
      State or province
    </ParamField>
    <ParamField body="postalCode" type="string" required>
      Postal or ZIP code
    </ParamField>
    <ParamField body="country" type="string" required>
      ISO country code (e.g., US, GB, DE)
    </ParamField>
  </Expandable>
</ParamField>

## Supported Tokens

| Token | Network | Minimum Amount | Processing Time |
|-------|---------|----------------|-----------------|
| SOL   | Solana  | 0.1 SOL       | 1-2 seconds    |
| USDC  | Solana  | 1 USDC        | 1-2 seconds    |
| USDT  | Solana  | 1 USDT        | 1-2 seconds    |
| BTC   | Bitcoin | 0.001 BTC     | 10-60 minutes  |
| ETH   | Ethereum| 0.01 ETH      | 2-15 minutes   |

## Card Networks

- **Virtual Cards**: Visa network with global acceptance
- **Physical Cards**: Available with Visa or Mastercard
- **3D Secure**: Supported for enhanced security
- **Contactless**: NFC payments enabled on physical cards

## Processing Times

- **Virtual Card**: Instant activation (1-2 seconds)
- **Physical Card**: 5-10 business days shipping
- **Funding**: Real-time for Solana tokens, varies for BTC/ETH
