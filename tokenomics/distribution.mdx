# Token Distribution Model

The ENTRO token distribution is designed to ensure long-term sustainability and fair allocation across all stakeholders.

## Creator Fees Buyback

Protocol implements systematic buyback mechanism:
- **15-20% of all protocol revenue** allocated for token buybacks
- Automatic market purchases create consistent demand
- Purchased tokens distributed to stakers and ecosystem
- Transparent on-chain execution
