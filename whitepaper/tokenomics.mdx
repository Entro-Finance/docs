## Utility & Value Accrual

### Primary Utilities
- **Governance**: Vote on protocol upgrades and parameter changes
- **Staking**: Earn rewards by securing the protocol
- **Yield Boosting**: Enhanced rewards in liquidity pools
- **Card Benefits**: Bonus value when funding cards with crypto

### Value Accrual Mechanisms
- **Creator Fees Buyback**: 15-20% of protocol revenue used for automatic token buybacks
- **Staking Rewards**: Enhanced yields from buyback distribution
- **Utility Growth**: Expanding use cases drive organic demand
- **Sustainable Model**: Revenue-aligned value creation
