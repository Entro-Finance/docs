# Merchant Integration

## Why Accept Entro Cards? 💳

**Lower Fees**: 0.5% vs 2-3% traditional processing  
**Instant Settlement**: Get funds immediately, not T+2  
**No Chargebacks**: Blockchain finality = no disputes  
**Global Customers**: Accept payments from 180+ countries  

## Integration Options

### 🔌 API Integration
```javascript
const entroMerchant = new EntroMerchant({
  apiKey: 'merchant_key_xxx',
  webhookUrl: 'https://store.com/webhook'
});

entroMerchant.on('payment', (tx) => {
  console.log(`Received $${tx.amount}`);
});
```
