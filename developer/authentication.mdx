# Authentication

## API Keys üîë

All API requests require authentication using API keys.

### Getting Your Keys

1. Sign up at [developers.entrofinance.com](https://developers.entrofinance.com)
2. Create a new project
3. Generate API keys

### Using API Keys

```bash
curl https://api.entrofinance.com/v1/cards \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript
const entro = new EntroClient({
  apiKey: process.env.ENTRO_API_KEY,
  apiSecret: process.env.ENTRO_API_SECRET
});
```

## Webhook Signatures

Verify webhook authenticity:

```javascript
const crypto = require('crypto');

function verifyWebhook(payload, signature) {
  const hash = crypto
    .createHmac('sha256', WEBHOOK_SECRET)
    .update(payload)
    .digest('hex');
    
  return hash === signature;
}
```
