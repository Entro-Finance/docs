## Quick Start

```javascript
import { EntroClient } from '@entrofinance/sdk';

const entro = new EntroClient({
  apiKey: 'sk_test_xxx',
  network: 'devnet'
});

// Create a card
const card = await entro.cards.create({
  type: 'VIRTUAL',
  currency: 'USD'
});

console.log(card.number);
```

## Core Methods

### Cards
```javascript
// List cards
const cards = await entro.cards.list();

// Get card details
const card = await entro.cards.get('card_123');

// Freeze card
await entro.cards.freeze('card_123');
```

### Transactions
```javascript
// Get transactions
const txs = await entro.transactions.list({
  cardId: 'card_123',
  limit: 10
});
```
